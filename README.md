# Tesis-Evaluacion-de-Seguridad-Estacion-de-Recarga-de-Hidrogeno
Se realizó un estudio cualitativo y cuantitativo sobre los principales modos de falla que se pueden encontrar en una hidrogenera y se seleccionó una de las fallas más críticas para desarrollar un análisis de Montecarlo y modelación del comportamiento de la falla, todo ello con MATLAB.
