# Evaluación de Seguridad y Predicción del Comportamiento de Modos de Fallas Críticos de una Estación de Recarga de Hidrógeno

## Descripción

Se realizó un estudio cualitativo y cuantitativo sobre los principales modos de falla que se pueden encontrar en una hidrogenera y se seleccionó una de las fallas más críticas para desarrollar un análisis de Montecarlo y modelación del comportamiento de la falla, todo ello con MATLAB.

## Objetivo

Crear un modelo de evaluación de la seguridad y los riesgos presentes en el funcionamiento seguro de una estación de recarga de hidrógeno combustible tipo.

## Herramientas

- MATLAB
- Google Scholar
- ScienceDirect
- Excel
- Draw.io

## Metodología

- Recolección de datos
- Limpieza de información
- Modelado matemático
- Validación

## Resultados

- Identificación de los modos de falla de mayor criticidad.
- Correlación entre fallas físicas y químicas.
